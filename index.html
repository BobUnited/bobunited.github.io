Personal Projects

Infini Run

An infinite runner game where you dodge fireballs and arrows by jumping, shrinking, floating, and flipping gravity. Along the way you pick up coins and powerups, allowing you to buy items in the shop. Created using C# and Unity. This was my first major Unity project. Click here for an online demo, or here for the GitHub repo.

Swords and Soldiers III

A game inspired by the indie game Swords and Soldiers II. Players try to destroy each other's castle by sending units and spells at each other in 2D combat. Can be played against the AI or over the internet by taking advantage of Photon Networking. Created using C#, Unity, and Photon. Click here for an online demo, or here for the GitHub repo.

Untitled Roguelike

A deckbuilding roguelike in the vein of games like Slay the Spire, with the twist that you can add monsters to your deck after you defeat them. Level up both yourself and your monsters to add more cards to your deck. Created using C# and Unity.

Untitled RPG

A 3D action RPG with lots of procedural generation. Basic terrain structure generated using Voroni polygons. Tree/rock/etc. placement uses a modified Poisson disc distribution. Towns and roads generated using a modified algorithm from this paper.
